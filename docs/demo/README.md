# Demo de vídeo / Demo-Video

## Español

Este directorio contiene el material necesario para grabar la demo oficial de Cimiento v1.0.

Archivos previstos:

- `guion-video-demo.md` — guion escena por escena en español y alemán.
- `video.mp4` — archivo final exportado. **No se crea en este commit**; queda reservado para cuando el usuario lo grabe.

Herramientas recomendadas en Linux:

- **OBS Studio** para grabación principal de pantalla y audio.
- **Peek** para clips cortos o GIFs auxiliares.
- **kdenlive** para edición básica, cortes, zooms y subtítulos.

Proceso recomendado:

1. Preparar un proyecto de demostración limpio con IFC ya generable.
2. Seguir el guion escena por escena sin improvisar entradas distintas.
3. Grabar la voz en off o, alternativamente, generar subtítulos a partir del texto literal del guion.
4. Exportar la pieza final a `docs/demo/video.mp4`.
5. Sustituir en el README el marcador `[VIDEO_DEMO_AQUI]` por el enlace real cuando el vídeo exista.

## Deutsch

Dieses Verzeichnis enthaelt die Unterlagen fuer die offizielle Demoaufnahme von Cimiento v1.0.

Geplante Dateien:

- `guion-video-demo.md` — Drehbuch Szene fuer Szene in Spanisch und Deutsch.
- `video.mp4` — finale Exportdatei. **Sie wird in diesem Commit nicht angelegt**, sondern spaeter manuell hinzugefuegt.

Empfohlene Werkzeuge unter Linux:

- **OBS Studio** fuer die eigentliche Bildschirmaufnahme und Audio.
- **Peek** fuer kurze Clips oder kleine GIF-Sequenzen.
- **kdenlive** fuer einfache Nachbearbeitung, Schnitte, Zooms und Untertitel.

Empfohlener Ablauf:

1. Ein sauberes Demo-Projekt vorbereiten, bei dem IFC und Render reproduzierbar funktionieren.
2. Das Drehbuch Szene fuer Szene abarbeiten.
3. Voice-over aufnehmen oder alternativ Untertitel direkt aus dem Skript ableiten.
4. Die fertige Datei als `docs/demo/video.mp4` exportieren.
5. Den Platzhalter `[VIDEO_DEMO_AQUI]` im README erst dann durch den echten Link ersetzen.
